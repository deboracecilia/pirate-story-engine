# 🏴‍☠️ Interactive Pirate Story — Python Text Adventure

Uma aventura interativa inspirada no universo de piratas, onde **você é o protagonista** da história.
Cada decisão molda o rumo da jornada, transformando o código em uma experiência dinâmica e envolvente ⚡

---

## 🎯 Sobre o projeto

Este projeto é um **jogo narrativo em Python**, desenvolvido para praticar lógica de programação de forma criativa.

O usuário:

* Cria seu próprio herói 🧑‍🚀
* Define habilidades e poderes 💥
* Escolhe caminhos e decisões 🧭
* Enfrenta batalhas e desafios ⚔️

Tudo isso através de **inputs no terminal**, tornando a experiência totalmente personalizada.

---

## 🧠 Conceitos aplicados

✔️ Variáveis
✔️ Entrada de dados (`input`)
✔️ Saída de dados (`print`)
✔️ Concatenação de strings
✔️ Estruturas condicionais
✔️ Estrutura narrativa interativa
✔️ Lógica de decisão

---

## 🎮 Como funciona

1. O jogador define:

   * Nome do herói
   * Poder supremo
   * Nome do navio
   * Frase de efeito
   * Habilidade especial

2. A história começa em um cenário misterioso 🌊

3. Durante a narrativa:

   * Você toma decisões
   * Escolhe ataques
   * Descobre fraquezas
   * Enfrenta inimigos

4. O jogo possui **múltiplos caminhos**, como:

   * ⚡ Ilha do Relâmpago Eterno
   * 🌊 Coração da Grand Line

---

## ⚔️ Destaques do jogo

## 🧪 Exemplo de experiência

```bash
Digite o nome do seu heroi: Débora
Digite o nome do seu ataque supremo: Chama Celestial
Escolha o nome do seu navio: Eclipse Negro
Escolha uma frase de efeito: O mar nunca esquece!
```

➡️ A partir disso, a história se adapta totalmente às suas escolhas.

---

## 🚀 Como executar

1. Instale o Python (se ainda não tiver)
2. Salve o código em um arquivo `.py`
3. Execute no terminal:

```bash
python nome_do_arquivo.py
```
